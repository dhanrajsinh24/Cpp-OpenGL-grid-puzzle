# C++ OpenGL Grid Puzzle Game - Queens (linkedin)

Queens is a logic game where you need to fill the grid so that there is only one Queen per row, column, and colored region.

- [Prerequisites](#prerequisites)

### Prerequisites
- OpenGL
- MinGW-w64
- CMake
- FreeGLUT
- GLEW
